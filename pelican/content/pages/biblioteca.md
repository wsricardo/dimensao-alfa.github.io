## Como a biblioteca está organizada

A biblioteca do Dimensão Alfa foi pensada como uma camada de curadoria, não apenas como uma lista de links. Cada entrada pode representar um artigo interno, uma lista de exercícios, uma trilha de leitura ou um material recomendado com contexto suficiente para orientar o estudo.

### Critérios de organização

- **Tema**: para aproximar materiais de uma mesma área.
- **Tipo**: para distinguir artigo, exercício, trilha, guia ou nota comentada.
- **Nível**: para orientar a entrada do leitor no acervo.
- **Idioma e autoria**: para facilitar busca e uso responsável do material.

## Como usar esta seção

Use a busca para encontrar termos específicos e depois refine por tipo, tema, nível ou idioma. Cada material tem uma ficha própria com contexto editorial, indicação de uso e links de saída quando existirem.
